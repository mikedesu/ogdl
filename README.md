tl;dr: downloads videos or music from youtube, ad-free, with a single command,
without knowing the Youtube URL for the video.

I had this idea real quick and I figured I'd hammer it out before I got 
something to eat.

Requirements:
- OSX/Linux/whatever gives u a shell
- Python 2.7
- youtube-dl and its requirements (ffmpeg, etc)
- a Google Youtube Data API Key: [http://console.developers.google.com/](http://console.developers.google.com/)

Usage:
./ogdl "Search Query"
./ogdl-music "Search Query"

ogdl will download the first video found for the search query in the best 
format available.

ogdl will download the first video found for the search query in the best 
format available, THEN strip the audio from the video and encode it in 
Ogg Vorbis format.
